Simplatab App

Please ensure Docker Desktop is installed and running before proceeding.

Please select the folder that contains the Train.csv and Test.csv files. These files should include: 1. A column named ID or patient_id which is the id of a row in the .CSV files. 2. The target column should be named "Target" and it should contain 0s and 1s (binary classification at the moment).


Please select the folder where you want to save the output files generated by the application.
